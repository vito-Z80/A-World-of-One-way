Каждый объект должен содержать init и update процедуры

init:
	Вызывается при построении комнаты если в объектах комнаты обнаружен ID текущего объекта.
	В .oData добавляются все необходимые параметры текущего объекта.

update:
	Вызывается каждый кадр для исследования содержимого объекта .oData .
	Опираясь на ранее созданные данные объекта .oData можно манипулировать объектом. 
	Передвижение объекта, смена цвета, анимация появления/уничтожения, и т.д.
